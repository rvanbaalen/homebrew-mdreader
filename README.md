# homebrew-mdreader

Homebrew tap for [mdreader](https://github.com/rvanbaalen/mdreader) — a beautiful macOS markdown reader.

## Install

```bash
brew install --cask rvanbaalen/mdreader/mdreader
```

## About

mdreader is an open source macOS app for reading markdown files with proper typography, syntax highlighting, and a design that gets out of the way.

See the [main repository](https://github.com/rvanbaalen/mdreader) for documentation, screenshots, and source code.
